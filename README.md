# Image-Processing
AMIGO Robot "For Kids Breeding based on Computer Vision" captures photos then analyzing them  through three stages "first: face detection to make sure there is person in picture, second: face  recognition to recognize a child who it takes care, third: emotion  recognition". Depending on results  of emotions recognition the robot behaves. For example, if the robot finds the child angry, it will deal with him by the ideal ways of child rearing (It will converse with him by inter active listening.
